Diwali Sales Data Analysis

📊 Overview

Analysis of Diwali sales data to understand customer purchasing behavior and sales trends using Python.

🛠️ Tools Used

 - Python
 - Pandas
 - NumPy
 - Matplotlib
 - Seaborn
 - Jupyter Notebook

📈 Project Goals

 - Clean and explore the dataset
 - Analyze customer buying patterns
 - Identify top-selling products
 - Visualize sales trends

📶 Insights

 - Top states contributing to sales
 - Gender-wise purchasing behavior
 - Popular product categories

🚀 How to Run

 - Open the notebook in Jupyter Notebook
 - Run all cells to see the analysis
